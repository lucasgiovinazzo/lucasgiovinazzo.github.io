📋 Cómo Cambiar de Tema

  1. Abrí index.html en un editor de texto
  2. Buscá la línea que dice: <link rel="stylesheet" href="css/styles.css">
  3. ENCIMA de esa línea, agregá el tema que querés (ver ejemplos arriba)
  4. Guardá el archivo y refrescá el navegador
  5. Repetí el proceso en proyectos/ispm.html

Tip: Probá los 3 temas, mostráselos a amigos, y elegí el que mejor te represente. El tema claro es más tradicional, el oscuro es moderno y profesional, y el mint es único y llamativo.

Recomendación: Para clientes institucionales/empresas → CLARO. Para startups/tech → OSCURO. Para destacar como developer → MINT.
